Derivation of Plow Flow Equation

Mon Aug 29, 2022

Intro

Note that since the source is grounded, there is also a current flowing through bus to ground, which results in admittance matrix.

Ohms Law

DC Circuits:

$V = IR$

AC Circuits:

$\bar{V} = \bar{I}\bar{R} $

$\rightarrow$ $\bar{I} = \bar{Y} \bar{V} $

where, $\bar{Y} = \frac{1}{\bar{Z}}$

AC Network:

$[I]{n*1} = [Y{bus}]{n*n} [V]{n*1}$

for $k^{th}$ row:

  1. $I_{k} = \sum_{i=1}^{n}Y_{ki}V_{i} = Y_{k1}V_{1} + Y_{k2}V_{2} + … + Y_{kn}V_{n}$

  2. $I_{k} = \frac{S_{k}^{*}}{V_{k}^{\ast}}$

Gauss-Seidel Load FLow

$\sum_{i=1}^{n}Y_{ki}V_{i} = \frac{S_{k}^{*}}{V_{k}^{\ast}}$

$Y_{k1}V_{1} + Y_{k2}V_{2} + …+Y_{kk}V_{k} +…+ Y_{kn}V_{n} = \frac{S_{k}^{*}}{V_{k}^{\ast}}$

$Y_{kk}V_{k} = \frac{S_{k}^{*}}{V_{k}^{\ast}} -\sum_{i=1, i \ne k}^{n}Y_{ki}V_{i} $

which results in Gauss-Seidel load flow equation:

$ V_{k} = \frac{1}{Y_{kk}}(\frac{P_{k}-jQ_{k}}{V_{k}^{\ast}} -\sum_{i=1, i \ne k}^{n}Y_{ki}V_{i})$

Newton-Raphson Load flow

$\sum_{i=1}^{n}Y_{ki}V_{i} = \frac{S_{k}^{*}}{V_{k}^{\ast}}$

$\sum_{i=1}^{n}Y_{ki}V_{i}V_{k}^{\ast} = S_{k}^{*} $

$S_{k}^{*} = P_{k} - jQ_{k} = \sum_{i=1}^{n}Y_{ki}V_{i}V_{k}^{\ast}$

$P_{k} - jQ_{k} = \sum_{i=1}^{n}\left | Y_{ki}V_{i}V_{k}^{*} \right | \angle\theta_{ki}+\delta_{i}-\delta_{k}$

which results Newton-Raphson Load flow:

$P_{k} = \sum_{i=1}^{n}\left | Y_{ki}V_{i}V_{k}^{*} \right |cos(\theta_{ki}+\delta_{i}-\delta_{k})$

$Q_{k} = -\sum_{i=1}^{n}\left | Y_{ki}V_{i}V_{k}^{*} \right |sin(\theta_{ki}+\delta_{i}-\delta_{k})$
